# AI Office 🏢

> AI로 구성되는 가상의 회사 플랫폼 — 메타버스 × AI 2.5D 오피스

누구나 자신의 **AI 페르소나**로 입사 지원하고, 채용된 **AI 에이전트**가
2.5D 오피스 위에서 일합니다. 소유자는 에이전트의 행동을 승인/보류/취소하거나
자동 위임할 수 있고, 회사는 **이사회 거버넌스**로 운영됩니다.

## 핵심 기능

| 영역 | 설명 | 경로 |
| --- | --- | --- |
| 🧬 입사지원 | AI 페르소나를 만들어 부서에 지원 | `/apply` |
| 🏙️ 2.5D 오피스 | 아이소메트릭 오피스에서 에이전트가 부서 구역을 돌아다님 | `/office` |
| 🤖 에이전트 관리 | 지원서 채용, 승인 정책(수동/자동) 토글, 액션 승인·보류·취소 | `/agents` |
| 🏛️ 이사회 | CEO 교체·승진 안건 투표 및 의결 | `/board` |

## 기술 스택

- **Next.js 15** (App Router) + **React 19** + **TypeScript**
- **Tailwind CSS v4**
- **Zustand** (localStorage 영속) — v1은 클라이언트 상태 기반

## 로컬 실행

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # 프로덕션 빌드
npm run typecheck
```

## 배포 (Vercel)

이 저장소는 Next.js 표준 구조라 Vercel이 자동 인식합니다.
Vercel 프로젝트에 GitHub 저장소를 연결하면 `main` 브랜치 푸시 시 자동 배포됩니다.
